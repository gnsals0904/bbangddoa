## 기능
### 1. help command
<div>
<img src="./img/helpcommand.gif" width="50%" height="50%"/>
</div>

#### `!help` 를 입력하면 봇이 사용자에게 빵또아 디스코드 봇 사용설명 방법을 알려줍니다.

<br>

### 2. clear command
<div>
<img src="./img/clearcommand.gif" width="50%" height="50%"/>
</div>

#### `!clear n` 를 입력하면 봇이 채널내의 n개의 메시지를 삭제합니다.

<br>

## 3. ping command
<div>
<img src="./img/pingcommand.gif" width="50%" height="50%"/>
</div>

#### `!ping` 를 입력하면 봇이 pong이라고 대답합니다.

<br>

## 4. search lol record command
<div>
<img src="./img/searchcommand.gif" width="50%" height="50%"/>
</div>

#### `!전적검색 닉네임` 을 입력하면 봇이 해당 닉네임으로 전적검색을 하고 사용자에게 해당 유저의 정보를 알려줍니다.

<br>
